User list

A list of users

GET /users{?name,joinedBefore,joinedAfter,sort,limit}

get users

Get a list of users. Example: 

                
https://api.mywebsite.com/users?sort=joined&limit=5

Parameters

name

string (optional) Example: 68a5sdf67

Search for a user by name

joinedBefore

string (optional) Example: 2011-01-01

Search by join date

joinedAfter

string (optional) Example: 2011-01-01

Search by join date

sort

string (optional) Default: name Example: joined

Which field to sort by

Choices: name joined -joined age -age location -location plan -plan

limit

integer (optional) Default: 10 Example: 25

The maximum number of users to return, up to 50

Response 200

Headers 

                
Content-Type: application/json

Body 

                
[
    {
        "name": "alice",
        "image": "http://foo.com/alice.jpg",
        "joined": "2013-11-01"
        },
        {
        "name": "bob",
        "image": "http://foo.com/bob.jpg",
        "joined": "2013-11-02"
    }
]

Schema 

                
{
    "type": "array",
    "maxItems": 50,
    "items": {
        "type": "object",
        "properties": {
            "name": {
                "type": "string"
            },
            "image": {
                "type": "string"
            },
            "joined": {
                "type": "string",
                "pattern": "\d{4}-\d{2}-\d{2}"
            }
        }
    }
}
